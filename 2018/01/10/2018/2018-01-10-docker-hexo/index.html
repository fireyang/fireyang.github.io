docker 化的 hexo

发表于 2018-01-10 更新于 2024-01-13 分类于 hexo

dockerfile

配置一个自己的 dockerfile，并提交到 docker
hub 上，这样以后在不同机器上，只要支持 docker 就可以写 blog 了

github: https://github.com/fireyang/hexo-docker
docker hub: https://hub.docker.com/r/fireyang/hexo/
